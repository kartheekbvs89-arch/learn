PROMPT 1 — Replace the "Content Coming Soon" placeholder with a real content data structure.

The lesson page currently shows "Content Coming Soon" for every lesson because there is no content data. Do the following:

1. Create a new file: src/data/lessonContent.ts (or .js)

2. Define and export a TypeScript type called LessonContent:

type CodeExample = {
  title: string;           // e.g. "Minimal example"
  filename: string;        // e.g. "main.py"
  language: string;        // e.g. "python"
  code: string;            // the actual code as a multiline string
  explanation: string;     // 1-2 sentences explaining what this shows
}

type LabStep = {
  step: number;
  title: string;
  command: string;         // the bash/python command to run
  expectedOutput: string;  // what the student should see
  explanation: string;
}

type QuizQuestion = {
  question: string;
  options: string[];       // 4 options
  correctIndex: number;    // 0-3
  explanation: string;     // shown after answering
}

type LessonContent = {
  moduleSlug: string;      // e.g. "python"
  lessonSlug: string;      // e.g. "types-variables-operators"
  objective: string[];     // 3-5 bullet strings of what student will learn
  whyItMatters: string;    // 2-3 sentences: real-world relevance
  concepts: {
    title: string;
    body: string;          // prose explanation
    diagram?: string;      // ASCII art string (optional)
  }[];
  codeExamples: CodeExample[];
  commonErrors: {
    errorMessage: string;
    cause: string;
    fix: string;
  }[];
  labSteps: LabStep[];
  quiz: QuizQuestion[];
  resources: {
    title: string;
    url: string;
    type: 'docs' | 'article' | 'repo' | 'cheatsheet';
  }[];
}

3. Export a const: export const lessonContentMap: Record = {}
   The key format is: "moduleSlug/lessonSlug" e.g. "python/types-variables-operators"

   - Import lessonContentMap
   - Look up content by `${moduleSlug}/${lessonSlug}`
   - If found, render the full content
   - If not found, keep the "Coming Soon" placeholder (DO NOT REMOVE IT — it is a fallback)

   -   — link cards with type badge
